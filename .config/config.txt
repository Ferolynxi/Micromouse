Note: - •Arduino IDE 2.0 is perfect to install and legacy version is not needed anymore.
	•The blink example from examples of ESP32 in file option in Arduino IDE also works.
	•The official board manager for ESP32 boards is already available by Espressif Systems in the
	 boards manager and doesn't require to add any URL in Preferences for the boards manager.
	•The LED_BUILTIN is already defined in the official board manager and doesn't require to
	 explicitly define the value for it.
